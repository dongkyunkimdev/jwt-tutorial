# jwt-tutorial
### 1. 목표
Spring Security를 이용하여 JWT 인증과 인가 및 회원가입, 로그인 서비스를 구현

### 2. 프로젝트 환경
- IDE : IntelliJ
- ORM : Spring Data JPA
- auth : Spring Security, JWT
- DB : H2(in-memory)
